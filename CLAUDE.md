# claude-hooks

Collection of Claude Code hooks bundled as one plugin.

**Before adding or modifying a hook, read [AGENTS.md](AGENTS.md)** — it defines
the module contract (one self-contained folder under `core/`, shared helpers in
`lib/hook-io.mjs`, one wiring line in `hooks/hooks.json`, exit-code rules).

Overview and install instructions: [README.md](README.md).
